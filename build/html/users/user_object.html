User object¶

Each user belong to particular Application. User can have different bio, title, roles etc in different applications he belong to.

Field Description
_id Id of the User object
appId Id of application which user belongs to
accountId Id of user account.
picture Url to the user picture
fullName User full name
email User email address
bio User info
userName Internal user name, used for mentions
groups Associative array of groups where user participates
settings User notification settings and boolean values for onboarding
statistic User statistic inside his company
title User title
roles Array of user roles.
createdOn Date when user was created (signed up)
